Setup Pre-requisites for the RISC-V Board:

  • For enhanced security, create a dedicated user on your RISC-V board named gitlab-runner-user. This user should not have sudo (root) privileges. The GitLab Runner service and all its associated files will operate under this user.
  • The host computer running this setup (where this Flask application is hosted) must be able to access the RISC-V board via SSH using a private key. Ensure this private key is configured for authentication with the gitlab-runner-user on the RISC-V board.
